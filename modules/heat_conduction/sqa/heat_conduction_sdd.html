Heat Conduction Software Design Description

Introduction

The Software Design Description (SDD) for Heat Conduction is a description of the software product and serves as guideline that describes the architecture of the system and all of its parts.

Dependencies

The Heat Conduction application is developed using MOOSE and is based on various modules, as such the SDD for Heat Conduction is dependent upon the following documents.

  • Framework System Design Description
