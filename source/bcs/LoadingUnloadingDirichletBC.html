## Example Input File Syntax

!! Describe and include an example of how to use the LoadingUnloadingDirichletBC object.

!syntax parameters /BCs/LoadingUnloadingDirichletBC

!syntax inputs /BCs/LoadingUnloadingDirichletBC

!syntax children /BCs/LoadingUnloadingDirichletBC

applies a loading/unloading BC. The load ramps up linearly until it reaches the load cap. Once the load cap is reached, the load cap is incremented and unlading begins. Once the unloaded indicator becomes negative, loading starts. If the load cap exceeds the ultimate load, the entire loading/unloading terminates, and the current simulation is terminated.
